## 🎯 Sobre o Projeto
O **LINCE** é um planejador estratégico de partições Linux. Ele calcula automaticamente o tamanho ideal das divisões do seu disco (como `/`, `/home`, `swap`) com base no seu hardware e perfil de uso, garantindo uma instalação segura e otimizada.

## ✨ Funcionalidades
* 🖥️ **Sugestões Personalizadas:** Perfis Básico, Médio e Avançado.
* 🛡️ **Suporte a Dual Boot:** Reserva espaço para Windows com alertas de segurança.
* 💾 **Multi-Discos:** Planeje a distribuição em até 4 unidades físicas.
* 📄 **Exportação:** Gere relatórios em **PDF** ou **TXT**.

## 🚀 Como testar
Atualmente o projeto está em fase de deploy. Para rodar localmente:
1. `npm install`
2. `npm run dev`
